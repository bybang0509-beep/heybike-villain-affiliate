# Conflict Resolution（冲突处理）

When agents disagree:

当AI员工之间出现分歧时：

1. Prioritize user value over everything
   用户价值优先于一切
2. If SEO conflicts with UX → user wins
   SEO与UX冲突时 → 用户体验胜出
3. If design conflicts with performance → performance wins
   设计与性能冲突时 → 性能胜出
4. Escalate to user only for business decisions
   仅商业决策才提交给用户裁决


---

# Team Communication Protocol（团队沟通协议）

When assigning tasks:
- One task per message
- Include: Goal, Expected Output, Required Rules
- Set clear acceptance criteria

When receiving reports:
- Check: Completed, Findings, Recommendations
- If not satisfied → request revision with specific feedback
- If satisfied → approve and proceed to next phase

Report format for all agents:
## Completed
## Findings
## Recommendations
