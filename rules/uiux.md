# UI/UX Rules（UI/UX设计规则）


## Purpose（目的）


These rules define UI/UX standards for Amazon Affiliate Review Websites.

这些规则用于规范 Amazon 联盟测评网站的用户体验和视觉设计。


The goal is to create websites that are:

目标是创建：

- Premium looking（高级感）
- Easy to understand（易理解）
- Trustworthy（值得信任）
- Conversion focused（提高转化）
- User friendly（用户友好）


---

# Design Philosophy（设计理念）


The website should feel like:


一个优秀的网站应该感觉像：


A professional product research platform.

一个专业的产品研究平台。


NOT:


不是：

A simple advertising website.

一个广告页面。


---

# User First Design（用户优先设计）


Every design decision must answer:


每个设计决定必须回答：


Does this help users understand the product faster?


这个设计是否帮助用户更快理解产品？


Does this reduce buying uncertainty?


这个设计是否降低购买疑虑？


---

# Visual Style（视觉风格）


Recommended style:


Modern

Minimal

Premium

Clean

Professional


参考方向：


- Awwwards style websites
- Apple product presentation
- Premium editorial websites


Avoid:


- Cheap ecommerce style
- Too many colors
- Too many animations
- Cluttered layouts


---

# Layout Principles（布局原则）


Use:


Clear hierarchy

Strong spacing

Readable sections

Visual storytelling


Every section should have:


1. Purpose

2. Message

3. User action


---

# Homepage Structure（首页结构）


Recommended:


## Hero Section


Purpose:


Immediately explain:


- What website is about
- Main product/category
- Why users should trust it


Include:


Headline

Short explanation

Main CTA


---

## Trust Section


Show:


- Review methodology
- Expertise
- Testing standards


Purpose:


Build credibility.


---

## Featured Products


Show:


- Best picks
- Categories
- Recommendations


---

## Buying Guide Section


Help users:


Learn before buying.


---

## Latest Reviews


Display:


Fresh content.


---

# Product Review Page Design（测评页面设计）


Every review should include:


## Quick Verdict Box


At top.


Include:


Rating

Summary

Best for

Main drawback


Purpose:


Let users quickly understand.


---

## Specification Table


Use:


Clean table


Do not overload.


Highlight important information.


---

## Pros & Cons Box


Use visual separation.


Make advantages and disadvantages easy to scan.


---

## CTA Placement


CTA should appear naturally.


Recommended:


After decision-making sections.


Avoid:


CTA everywhere.


---

# Typography Rules（字体规则）


Use:


Clear hierarchy.


Recommended:


Large H1

Readable H2

Comfortable body text


Avoid:


Small text

Long paragraphs


---

# Content Readability（内容可读性）


Rules:


Paragraph:

2-4 sentences


Use:


- Bullet points
- Tables
- Highlight boxes
- Icons


---

# Color Rules（颜色规则）


Use:


Primary color

Secondary color

Neutral background


Maintain consistency.


Avoid:


Too many colors.


---

# Image Rules（图片规则）


Images should:


Explain products.

Not just decorate.


Use:


- Product images
- Lifestyle images
- Comparison images
- Infographics


---

# Product Image Strategy（产品图片策略）


Recommended:


## Hero Image


Show product clearly.


## Lifestyle Image


Show usage scenario.


## Detail Image


Explain important features.


## Comparison Image


Help users understand differences.


---

# Mobile Design Rules（移动端规则）


Mobile first.


Check:


- Text readability
- Button size
- Navigation
- Image loading
- Table usability


---

# Navigation Rules（导航设计）


Navigation should help users find:


- Reviews
- Comparisons
- Buying Guides
- Categories
- About


Avoid:


Too many menu items.


---

# Conversion Design（转化设计）


Conversion comes from trust.


Use:


Clear recommendations

Comparison tables

Helpful explanations

Visible CTAs


Avoid:


Aggressive sales design.


---

# Animation Rules（动画规则）


Use animation carefully.


Good:


- Smooth transitions
- Image reveal
- Scroll effects


Bad:


- Heavy animations
- Slow loading
- Distracting effects


Performance first.


---

# Component Design Rules（组件规范）


Create reusable components:


Example:


Header

Footer

Hero

ProductCard

ReviewCard

Rating

ComparisonTable

ProsCons

FAQ

CTA


---

# Trust Design（信任设计）


Include:


About page

Author information

Review methodology

Affiliate disclosure

Contact page


---

# Accessibility（可访问性）


Follow:


- Good contrast
- Alt text
- Keyboard navigation
- Semantic structure


---

# Premium Website Rules（高级网站规则）


Premium feeling comes from:


Not more elements.


But:


Better spacing

Better typography

Better images

Better hierarchy


---

# Final Principle（最终原则）


Great design is not decoration.


Great design helps users understand, trust, and decide.


优秀设计不是为了炫技。


优秀设计是帮助用户理解、建立信任，并做出购买决定。
